## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (v14 or higher)
- npm or yarn
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/franksaint2007-dev/bubblegum.git
cd bubblegum
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Set up your environment (if needed):
```bash
cp .env.example .env
# Edit .env with your configuration
```

### Usage

To get started with the project:

```bash
npm run dev
# or
yarn dev
```

This will start the development server. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
